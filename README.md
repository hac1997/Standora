## Rodando o projeto

1. Copie o arquivo de ambiente para `/web`

> Nota: caso o arquivo esteja como `env`, renomeie para `.env`

2. Entre na pasta web

```bash
cd web
````

3. Instale as dependências

```bash
npm i
```

4. Gere o cliente Prisma

```bash
npx prisma generate
```

5. Rode o projeto

```bash
npm run dev
```
